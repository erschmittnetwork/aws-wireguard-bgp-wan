Featured Lab Project: AWS WireGuard BGP WAN with Automated HA Hub Failover
Project Description
Technical documentation of an AWS-hosted hub-and-spoke WAN lab using WireGuard for encrypted overlay transport, FRRouting (BGP) for dynamic routing control, and AWS-native automation for active/standby hub failover.
Architecture Principles
Models enterprise WAN behavior: deterministic BGP path selection, encrypted overlays, automated failure handling, and operational observability under realistic SME constraints.
Technical Outcome
Two AWS EC2 hub nodes and multiple local VM spokes provide WireGuard-secured connectivity, dynamic route propagation, and automated hub failover without proprietary SD-WAN control planes.
Delivery Pipeline
Source documentation and diagrams are version-controlled in GitHub. CI/CD renders Mermaid diagrams, builds HTML/PDF artifacts with Pandoc, and publishes via S3 and CloudFront for deterministic delivery.
