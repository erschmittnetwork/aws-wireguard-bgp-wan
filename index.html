Flagship Project: AWS WireGuard BGP WAN with Automated HA Hub Failover
Project Focus
This project implements an AWS-hosted hub-and-spoke WAN using WireGuard for encrypted transport, FRRouting (BGP) for dynamic routing, and automated failover orchestration.
Architecture Principles
The lab models enterprise WAN design: deterministic BGP routing, encrypted transport, automated failure handling, and operational observability under realistic SME constraints.
Technical Outcome
Two AWS hub nodes and multiple remote spokes provide WireGuard-encrypted WAN connectivity, dynamic BGP routing, and automated hub failover without proprietary SD-WAN platforms.
Delivery Pipeline
Source documentation and diagrams are version-controlled in GitHub. CI/CD workflows render Mermaid diagrams, build HTML/PDF artifacts with Pandoc, and deploy via S3 and CloudFront for deterministic, repeatable delivery.
